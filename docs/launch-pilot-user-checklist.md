# Launch Pilot User Checklist

Use this checklist for controlled production launch pilot users.

## Before Starting

- [ ] Confirm you are using approved pilot data.
- [ ] Do not upload sensitive customer exports unless explicitly authorized.
- [ ] Use a clear assessment name.
- [ ] Report issues with screenshots and assessment ID.

## Account

- [ ] Create account.
- [ ] Sign in.
- [ ] Confirm dashboard opens.

## Assessment

- [ ] Create assessment.
- [ ] Confirm assessment appears in list.
- [ ] Open assessment.
- [ ] Save manual intake.
- [ ] Save cost/risk assumptions.
- [ ] Leave and return to confirm progress persists.

## Evidence

- [ ] Confirm upload is locked before prerequisites.
- [ ] Complete prerequisites.
- [ ] Upload approved evidence.
- [ ] Confirm evidence appears in history.
- [ ] Parse evidence if needed.
- [ ] Confirm inventory/risk does not crash.

## Report

- [ ] Open report preview.
- [ ] Review evidence received/missing.
- [ ] Request full report if needed.
- [ ] Wait for admin/manual entitlement.
- [ ] Generate/download PDF.
- [ ] Confirm PDF opens.

## Feedback

- [ ] Report unclear UI.
- [ ] Report errors.
- [ ] Report missing evidence messages.
- [ ] Report any redirect or login issue.
