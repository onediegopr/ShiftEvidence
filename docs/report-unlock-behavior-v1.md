# Report Unlock Behavior v1

## Free Preview
- Disponible sin unlock.
- Muestra resumen ejecutivo, tecnico, findings limitados y matrix preview limitada.

## Readiness Report unlocked
- Desbloquea la ruta comercial del full report.
- Permite generar `readiness_report` PDF v1.

## Pro unlocked
- Desbloquea la matrix completa y la experiencia pro visual.
- Sigue siendo preliminar.

## Storage Add-on unlocked
- Marca el addon de storage como activo.
- No implementa scoring profundo de storage todavia.

## Technical Review unlocked
- Habilita el estado visual de review.
- No existe scheduling real.

## PDF
- `free_preview` sigue existiendo.
- `readiness_report` se usa cuando el full report ya fue concedido.
- El PDF sigue siendo preliminar, no final.
