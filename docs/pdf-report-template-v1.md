# PDF Report Template v1

## Purpose
Create a readable, technical, preliminary PDF preview from the current assessment data.

## Sections
1. Cover
2. Executive Summary
3. Environment Summary
4. Cost / Risk Summary
5. Top Findings
6. VM Risk Matrix Preview
7. Missing Evidence
8. Locked Sections
9. Disclaimer

## Data used
- Assessment title and client label
- Report Preview data
- AssessmentScore
- RiskFindings
- ParsedInventorySummary
- Cost / Risk assumptions
- Evidence status

## Limitations
- Not a final report
- Not certified
- Not a paid unlock
- Not a migration plan
- Not a migration execution artifact

## Disclaimer
The PDF explicitly states that it is a preliminary readiness preview and not final engineering validation.
